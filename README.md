# Coin Collector Game 🎮

A single-stage 2D/3D coin collection game developed using Unity and C#. The objective of the game is to collect all coins scattered across the level while avoiding enemies. Players must strategically navigate obstacles and survive enemy encounters to successfully complete the stage.

## 🚀 Features

- Coin collection-based gameplay system
- Enemy avoidance mechanics
- Single-level progression
- Player movement and collision system
- JSON-based data handling
- Interactive game environment
- Win condition after collecting all coins

## 🛠️ Tech Stack

- **Game Engine:** Unity
- **Programming Language:** C#
- **Data Handling:** JSON
- **IDE:** Visual Studio

## 🎯 Gameplay

The player must:
1. Navigate through the environment.
2. Avoid enemy collisions.
3. Collect all available coins.
4. Complete the level after gathering every collectible.

## 📂 Project Structure

```bash
Assets/
│── Scripts/          # C# gameplay scripts
│── Prefabs/          # Game objects and reusable assets
│── Scenes/           # Main game scene
│── Resources/        # JSON and game data
│── Materials/        # Visual assets
```

## ⚙️ Installation & Setup

1. Clone the repository:
```bash
git clone <your-github-repo-link>
```

2. Open the project in Unity Hub.

3. Launch the project using a compatible Unity version.

4. Click **Play** in the Unity Editor to start the game.
